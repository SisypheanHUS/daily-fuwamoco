# Daily FUWAMOCO

<img src="docs/screenshots/glass-redesign-demo.gif" alt="Demo — morning greeting, then Home, Rituals, Calendar, Settings" width="280" />

Daily FUWAMOCO is a cozy companion app inspired by FUWAMOCO and, especially,
Mococo's heartfelt pep talks that somehow always arrive at the right time.

A small place to start the day with a warm greeting, a few encouraging
words, a gentle streak, and tiny everyday rituals with two mascots that
never ask you to be more productive — only to keep going.

Inspired by [FUWAMOCO](https://www.youtube.com/channel/UCt9H_RpQzhxzlyBxFqrdHqA)
— [Fuwawa](https://hololive.hololivepro.com/en/talents/fuwawa-abyssgard/) and
[Mococo](https://hololive.hololivepro.com/en/talents/mococo-abyssgard/)

## Features

- Morning greeting with a voice clip, wallpaper, and quote of the day
- Streak tracking for daily opens
- Rituals — small recurring habits with their own streaks
- Calendar view of what you did each day
- Morning check-in / evening reflection
- Collectible charms that unlock as your streak grows
- Fully offline, no account needed

## Install & run

```powershell
flutter run -d chrome              # open in a browser window
```

That's it — everything runs locally in your browser, no account or backend needed.
